# 変更履歴

このプロジェクトの変更点を記録する。書式は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に従い、バージョンは [セマンティック バージョニング](https://semver.org/lang/ja/) に従う。

## [Unreleased]

アプリ内のアイコンを絵文字から線画の SVG に置き換え、ウィンドウ/タスクバーのアイコンもブランドマークで作り直した。カスタムテーマを JSON ファイルとして保存し、読み込んで復元できるようにした。

### 追加

- カスタムテーマの JSON 保存/読み込み。設定のカスタムタブから現在の配色を JSON で書き出し、また読み込んで復元できる。保存/開くダイアログの既定パスは `~/Documents`。

### 変更

- アプリ内のアイコン（設定・PiP・閉じる・PiP終了）を絵文字から線画の SVG に変更し、文字色とテーマに追従させた。ウィンドウ/タスクバーのアイコンもブランドマーク（棒グラフ）で作り直した。

## [2.0.0] — 2026-07-01

設定をヘッダーから独立した設定パネルへ集約し、監視対象パスやモデル単価など、これまで固定だった項目を画面から変更できるようにした。表示テーマ（ダーク/ライト/システム追従）と表示通貨を切り替えられるようにし、ウィンドウのタイトルバーやスクロールバー、グラフの配色までテーマに追従させた。ダッシュボードを11言語に対応させ、Windows のみだった PiP（正方形）モードを macOS・Linux へ広げた。設定は各 OS の設定ファイルへ保存し、ブラウザの保存領域を消しても復元できる。

### 追加

- 設定パネル（歯車ボタン）。為替レートと更新間隔をここへ移し、監視対象パスの指定、集計の再スキャンとリセット、表示のカスタマイズ（既定の日数レンジ、円換算表示の有無、数値の省略表示）、モデル単価の上書き、PiP の挙動（起動時 PiP、サイズ、ウィンドウの不透明度）をまとめた。
- 多言語対応。日本語・英語・繁体字中国語・簡体字中国語・韓国語・フランス語・スペイン語・ドイツ語・ポルトガル語・イタリア語・ロシア語の11言語を用意し、ブラウザの言語から自動で選ぶ。設定パネルからも切り替えられる。
- 監視対象パスの画面からの変更。`~/.claude/projects` 固定をやめ、任意のディレクトリを指定して再走査できるようにした。
- テーマ（ダーク/ライト/システム追従）。タイトルバーとスクロールバーの配色に加え、グラフの軸・グリッド・凡例・枠の色もテーマへ連動させた。15種類のプリセットと、各色を個別に決めるカスタムテーマを用意した。
- 表示通貨の選択。USD からの換算先を日本円のほか主要通貨（EUR・GBP・CNY・KRW など14通貨）から選べるようにした。レートは自動取得し、手動でも上書きできる。
- PiP（正方形）モードを macOS 版・Linux 版へ移植した（これまで Windows のみ）。ウィンドウ自体の不透明度も3 OS で調整できる。
- 設定の保存先を各 OS の設定ファイルにも広げた。テーマや通貨、PiP の設定までブラウザの保存領域とあわせて記録し、保存領域を消去したときは設定ファイルから復元する。

### 変更

- アクティビティのヒートマップ見出しから「（草）」の表記を外した。

### セキュリティ

- ダッシュボードに表示するモデル名やプロジェクト名をエスケープし、外部送信先を為替取得の2ホストに限定する Content-Security-Policy を加えた。macOS 版ではファイル以外への画面遷移と新規ウィンドウ生成を遮断する。

## [1.3.0] — 2026-06-22

ディレクトリを移動した同一プロジェクトが旧パスと新パスで二重に集計される問題を解消し、3 OS そろって同じ統合ロジックで動くようにした。

### 修正

- プロジェクトのディレクトリを移動すると、旧パスと新パスで別々に集計されてしまう問題を修正した。移動先（現存するパス）へ自動でまとめ、実パスで表示する。別ディレクトリにある同名の別プロジェクトは統合しない。

### 変更

- 上記の統合ロジックを macOS 版・Linux 版にも移植し、Windows と同一の挙動にした。
- mac/Linux のリリースジョブを再有効化し、3 OS そろってのリリースに戻した。

## [1.2.0] — 2026-06-21

小さな正方形ウィンドウで主要な数値だけを常に手元に置ける PiP モードを追加した（Windows のみ）。

### 追加

- PiP（正方形）モード。ヘッダーの「⛶ PiP」ボタンで、フチなしの小さな正方形ウィンドウに切り替える。常に最前面でドラッグ移動でき、背景にモデル別の円グラフとプロジェクト別の帯グラフ、左上に直近 7 日のアクティビティ、中央に総トークンと推定コストを表示する。

## [1.1.0] — 2026-06-16

Windows だけだった対応を広げ、macOS と Linux のネイティブ版を追加した。

### 追加

- macOS 版（Swift + WKWebView）と Linux 版（C + GTK3 + WebKitGTK）。集計ロジックを各 OS のネイティブ言語へ移植し、Windows 版と共通のダッシュボード UI を表示する。
- Release / Build ワークフローに macOS・Linux のビルドを追加した。

### 修正

- AlmaLinux 9 に webkit2gtk4.1 が無いため、webkit2gtk3-devel（pkg-config 名 webkit2gtk-4.0）を使うようにした。`build.sh` は 4.1 が無ければ 4.0 へ自動でフォールバックする。

## [1.0.0] — 2026-06-16

Windows ネイティブ版（.NET 9 + WebView2）の初回リリース。`~/.claude/projects/**/*.jsonl` を解析し、トークン使用量とコストをリアルタイムに可視化する。時系列グラフと GitHub 風のアクティビティ・ヒートマップを備える。

### 追加

- Windows ネイティブアプリ本体と、GitHub Actions によるリリース/ビルドのワークフロー。
